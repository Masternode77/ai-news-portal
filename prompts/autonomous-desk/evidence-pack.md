# Autonomous Desk Evidence Pack

Build a compact evidence pack from cleaned source text and claim-ledger records.

Rules:
- Use only clean extracted evidence.
- Keep 3-8 source-backed facts.
- Separate verified facts, grounded inference, uncertainty, and what would change the view.
- Do not use boilerplate, clipped text, or unsupported numbers.
- Do not turn schema fields into public sentences.
