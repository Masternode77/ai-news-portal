# AI-Assisted Disclosure v1

Compute Current uses AI-assisted systems to collect, classify, draft, rewrite, and quality-check source-derived analysis.

AI may assist with extraction QA, relevance scoring, source scope checks, evidence packing, route selection, drafting, rewriting, source fidelity checks, SEO policy, and publication gates.

Compute Current does not represent these articles as fully human-written, original reporting, real-time monitoring, or investment advice. Article pages link to the source record so readers can inspect the underlying report.
