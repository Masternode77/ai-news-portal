# Analytics Event Taxonomy

Tracking is no-op by default and pushes events to `window.dataLayer`. If configured, events are also sent to GA4 or Plausible.

## Required Environment Variables

- `PUBLIC_GA4_ID`
- `PUBLIC_PLAUSIBLE_DOMAIN`

## Events

- `newsletter_signup_view`
- `newsletter_signup_submit`
- `pricing_view`
- `pro_cta_click`
- `enterprise_cta_click`
- `advertise_cta_click`
- `media_kit_view`
- `sponsor_inquiry_submit`
- `report_view`
- `report_purchase_click`
- `directory_listing_request`
- `premium_article_preview_view`
- `paywall_cta_click`
