# Angle

Use this skill before drafting or regenerating analysis.

## Canonical Rules
- Every generated article needs a source-specific thesis.
- Do not repeat the same structure or framing across consecutive items.
- The angle should name the infrastructure decision point, not a generic AI market takeaway.

## Workflow
1. Extract the change, constraint, affected parties, exposure, and watch item.
2. Choose the clearest infrastructure angle from source facts.
3. Check nearby articles for repeated openings, structure, and rhythm.
